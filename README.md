# EnhancedCombatHUD-Dragonbane

An implementation of the Argon - Combat HUD for the Dragonbane system in Foundry VTT. The Argon - Combat HUD (CORE) module is required.

## Compatibility

- Foundry VTT v12-v13
- Argon Combat HUD (CORE)
- Dragonbane system
